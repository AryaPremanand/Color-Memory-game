# Color-Memory-game
A simon says type of color memory game - Players click the red button 3 times to turn the game on, the box will present 1 new color for you to remember with each level you surpass.
# Built with
- Arduino Uno + Full sized breadboard circuit
- 4 LED's (Red, Green< blue and yellow)
- 4 tactile quad-pin buttons
- 220Ohm Resistors
- jumper wires
- 3D Printed enclosure and button extensions (CAD & Ultimaker Slicer)
- Arduino IDE (C++ based programming software)
# How it works
1. Triple click the red button to turn on the game, all the lights will proceed to flash
2. The game will present you with a series of lights for you to repeat back on the buttons
3. Each level will add 1 more light to remember
4. When you make a mistake and lose the game, all the lights will flash and proceed to turn off completely
5. Triple click the red button to keep playing!
# Setup
The game is battery operated since the code is already loaded onto the arduino already. (No Computer needed to play)
